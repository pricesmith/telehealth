## To install and setup:

### install Ruby 2.7 on your computer

```
rvm install 2.7.0
```

```
bundle install
yarn
rake db:create
```

## To run:

```
bundle exec rails server
env PUBLIC_URL="" ./bin/webpack-dev-server
```

#### Card design page: http://localhost:3000/#/home/Design

####

#### Presentation page: http://localhost:3000/#/

Create a card on the card design page. After submission, view and cycle
through created cards on the presentation page
